# GH-600 Exam Prep Generator

Comprehensive exam preparation package generator for the GitHub Certified Agentic AI Developer (GH-600) certification.

## Overview

This system transforms source material from Microsoft Learn and related documentation into a complete, multi-phase study resource optimized for exam success. The final output is a static study website deployed to GitHub Pages via MkDocs Material.

## Installation

```bash
pip install -e ".[dev]"
```

## Usage

```bash
gh600-prep
```

## Development

```bash
pip install -e ".[dev]"
pytest
```
